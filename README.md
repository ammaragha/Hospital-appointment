## technologies
1- Larave (back end)
2- Angular (front end)

## How to run
- Back end 
```bash
$ cd ./server
```
$ composer install
$ cp .env.example .env
#### connect to your DB into .env file
$ php artisan key:generate
$ php artisan migrate: --seed


- Front end
$ cd ./client
$ npm install
$ ng serve
