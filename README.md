## technologies
1- Larave (back end)
2- Angular (front end)

## How to run
- Back end 
```bash
$ cd ./server
```
```bash
$ composer install
```
```bash
$ cp .env.example .env
```
connect to your DB into .env file
```bash
$ php artisan key:generate
```
```bash
$ php artisan migrate --seed
```

- Front end

```bash
$ cd ./client
```
```bash
$ npm install
```
```bash
$ ng serve
```
